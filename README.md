# Проект: Место

### Интерактивная веб страница, на которой отображены данные о человеке (фотография, имя, профессия), подборка фотографий.

### В проекте используются следующие технологии:

1. HTML
2. CSS

- Бэм
- Nestled Бэм
- Flex-box
- Grid Layout
- Интерактивные ссылки с hover
- Использование адаптивной верстки

3. Javascript

- Редактирование полей формы
- Форма добавления карточки
- Добавление карточки
- Лайк карточки
- Удаление карточки
- Открытие попапа с картинкой
- Плавное открытие и закрытие попапов кликом на крестик, оверлей, Esc
- Валидация форм

4. Figma
5. PerfectPixel - проверка верстки сайта
